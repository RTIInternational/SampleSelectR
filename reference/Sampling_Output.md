# Output sampling summary to the console Displays sample size, frame size, and optionally the sampling interval and random start.

Output sampling summary to the console Displays sample size, frame size,
and optionally the sampling interval and random start.

## Usage

``` r
Sampling_Output(n, N, k = NULL, r = NULL, curstrat = NULL)
```

## Arguments

- n:

  Sample size.

- N:

  Frame size.

- k:

  Optional sampling interval.

- r:

  Optional random start.

- curstrat:

  Optional stratum label.

## Value

Prints output to the console. Invisibly returns NULL
